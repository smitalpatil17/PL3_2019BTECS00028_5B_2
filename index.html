Program to converts binary number to decimal number

Code

            #include 
                int main()
                {
                    int bin, dec=0, i=1, dig;
                    cout << "Enter any Binary Number: " ;
                    cin >> bin;
                    while(bin!=0)
                    {
                        dig = bin%10;
                        dec = dec + (dig*i);
                        i = i*2;
                        bin = bin/10;
                    }
                    cout << "\nEquivalent Decimal Value = " << dec;
                    return 0;
                }

Sample input

Enter any Binary Number: 1100

Sample output

Decimal equivalent for binary number 1100 is 12

Variables used in the program

bin, dec, i, ,digit

Complexity of code

O(n) where n is number of digits in binary number
